# JavaScript – Funciones

Este proyecto contiene una serie de ejercicios prácticos enfocados en el uso de funciones en JavaScript, incluyendo funciones de expresión, arrow functions y manipulación del DOM mediante eventos.

Cada ejercicio se encuentra organizado en su propia carpeta y contiene los archivos necesarios para su funcionamiento.
